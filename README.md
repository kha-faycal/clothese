# clothese
AlphaShop est une plateforme e-commerce full-stack performante, optimisée pour le modèle de vente **Cash on Delivery (Paiement à la livraison)**. L'application intègre une interface client fluide et un panneau d'administration complet pour la gestion en temps réel des produits, des variantes complexes, des stocks et  suivi financier des commandes.
